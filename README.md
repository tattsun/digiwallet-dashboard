# digiwallet-dashboard

## Install

```
# in dw-client/
$ npm install


# in dw-server/
$ npm install
```

## Configuration

Place `dw-server/.env` as a configuration file for `dw-server` as below.

```
ETHERSCAN_API_KEY=XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
```

## Run

```
# in dw-client/
$ npm run dev


# in dw-server/
$ npm run dev
```
